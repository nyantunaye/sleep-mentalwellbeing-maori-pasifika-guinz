# Sleep Patterns and Mental Wellbeing in New Zealand Youth

## Overview
This project analyses the relationship between sleep patterns and mental wellbeing among Māori and Pasifika youth using longitudinal data from the Growing Up in New Zealand (GUiNZ) study.

## Objectives
- Analyse sleep patterns across ages 2–12  
- Examine ethnic differences in sleep behaviour  
- Investigate the relationship between sleep and mental wellbeing  
- Identify socioeconomic and household influences  

## Methods
- Linear regression  
- Mixed-effects models  
- Random forest  
- ANOVA and chi-square tests  

##  Key Findings
- Sleep declines as children grow older  
- Māori and Pasifika youth generally sleep less  
- Shorter sleep is linked to higher anxiety and depression  
- Psychological factors strongly influence sleep quality  

## Data Note
Original GUiNZ data cannot be shared due to restrictions. This repository demonstrates the analytical approach using reconstructed examples.
